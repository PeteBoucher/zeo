WebVR

Build persistent muiltiplayer VR worlds with Javascript.
Start the node server and drag 'n drop npm modules in the browser. Anyone can join with a URL.
You don't even need a headset! It works with mouse + keyboard.

Made for full-stack web developers

Build entity-component worlds without taking off your headset.
Code and debug with vanilla node and DevTools.
The API is DOM + THREE.js on the front and node on the back, so it works with pretty much everything.
